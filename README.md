# bm-dashboard
Dashboard operativa per il monitoraggio flussi di lavoro, turni e analisi finanziaria nel settore marittimo. Include parser per importazione dati, calcolo automatico ritenute forfettarie e tracker fatturazione. Progettata per Bottale Marine.
